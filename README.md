PetNet Engagement Predictor

This project explores engagement prediction in a social pet platform using exploratory data analysis and regression modeling.

Objectives

Understand factors influencing post engagement

Identify behavioral patterns in content performance

Build a predictive model for engagement estimation

Techniques Used

Exploratory Data Analysis (EDA)

Statistical correlation analysis

Linear Regression modeling

Feature impact interpretation

Key Insights

Audience size strongly drives engagement

Comments significantly increase interaction

Video duration has moderate impact

Posting time shows weak influence

Engagement distribution is stable without extreme outliers
